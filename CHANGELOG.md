# 更新紀錄 (CHANGELOG)

## [2026-03-13] - 重大更新：UI 強化與行動裝置優化

此版本將應用程式從傳統 HTML/JS 重構為現代 React 架構，並加入了大量增強導向的功能。

### 🌟 新增功能
- **整合式多軌介面**：混音軌現在直接整合在主音軌下方，雙軌時間軸完美對齊。
- **混合疊加預覽**：新增「顯示混音」按鈕，可將混音軌片段半透明地疊加在主音軌上同步檢視。
- **橘色定位游標**：在主音軌點擊會產生橘色游標，並支援使用方向鍵即時微調位置。
- **智慧尋找強化**：
    - 搜尋結果以綠色區塊高標示顯示。
    - 支援一次顯示所有搜尋目標或在目標間切換。
    - 尋找按鈕即時顯示符合數量，如 `尋找(22)`。
- **行動裝置模式**：可於設定開啟「行動模式 Modal 介面」，針對小螢幕將操作按鈕與搜尋欄位 Modal 化。
- **復原機制 (Undo/Redo)**：完整支援剪除、音量增強、保留選區等操作的回復與重做。
- **片段列表彈出化**：支援以快捷鍵（預設為 `x`）快速切換片段管理視窗，靈活釋放編輯空間。

### 🛠 修復與改進
- **快捷鍵衝突修復**：解決空白鍵會導致網頁捲動的問題，現在全域對應至播放/暫停。
- **持久化儲存優化**：改進設定儲存機制，支援深層合併以防新功能設定被舊緩存覆蓋。
- **視覺層級修復**：修正搜尋標記被 Canvas 背景遮擋的問題。
- **排版優化**：解決操作選單在窄螢幕或特定解析度下被遮擋裁切的問題。
