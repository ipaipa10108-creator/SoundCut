# SoundCut - 專業線上音訊剪輯工具

SoundCut 是一個基於 React 開發的高效能線上音訊剪輯應用程式。它提供了直觀的波形視覺化界面，並支援多軌混音、智慧片段搜尋、以及多種音訊處理功能。

## 🌟 核心功能

*   **實時波形視覺化**：高流暢度的 Canvas 渲染，支援縮放與平移。游標懸停時可即時顯示預估頻率 (Hz)。
*   **多軌編輯體驗**：整合主音軌與混音軌，兩者左右時間軸完美對齊，支援半透明疊加檢視。
*   **智慧尋找**：可根據「音量」或「頻率」自動偵測靜音段或特定聲音，並以綠色標記顯示。
*   **強大編輯工具**：
    *   **提取 (Extract)**：將選區存入片段清單。
    *   **音量增強 (Amplify)**：自訂倍率增強選區音量。
    *   **剪除 (Cutout)**：永久移除選區。
    *   **保留 (Keep)**：僅保留選取範圍。
*   **完整復原機制**：支援多層級 Undo (`Ctrl+Z`) 與 Redo (`Ctrl+Y`)。
*   **專業混音匯出**：將主音軌與多個安插片段合併，支援匯出為 MP3 或 WAV 格式。
*   **自訂快捷鍵**：可在設定介面自行配置操作熱鍵。
*   **行動裝置優化**：提供「行動模式 Modal 介面」，在窄小螢幕上也能精確操作。

## 🚀 快速開始

### 環境需求
*   Node.js (建議 v18 以上)
*   uv (Python 環境管理，若需要執行相關腳本)

### 安裝步驟
1. 複製專案：
   ```bash
   git clone <repository-url>
   cd SoundCut
   ```
2. 安裝套件：
   ```bash
   npm install
   ```
3. 啟動開發伺服器：
   ```bash
   npm run dev
   ```

## ⌨️ 預設快捷鍵

*   `空白鍵 (Space)`：播放 / 暫停主音軌選區。
*   `x`：開啟 / 關閉右側片段列表視窗。
*   `Ctrl + Z`：復原上一步。
*   `Ctrl + Y`：重做下一步。
*   `方向鍵 (←/→)`：微調目前播放/選取位置（暫停時）。
*   `1`, `1.5`, `2`：快速切換播放倍速。

## 🛠 關鍵技術
*   **前端框架**：React + TypeScript
*   **狀態管理**：Zustand (配合持久化儲存)
*   **樣式處理**：TailwindCSS
*   **音訊處理**：Web Audio API
*   **圖示庫**：Lucide React
